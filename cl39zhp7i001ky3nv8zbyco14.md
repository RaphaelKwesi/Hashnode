### Elastic Cloud Compute (EC2)
[Elastic Cloud Compute](https://www.amazonaws.cn/en/ec2/) is a part of Amazon Cloud Computing that provides servers for rent in the cloud. Elastic üèπ like the name... EC2 is a cloud service that grows or shrinks based on the number of servers needed to run a deployed application. The servers used in a cloud computing environment are called **instances**. EC2 instances are run with user-defined configurations.

EC2 are not considered serverless because instances are actual physical servers in data centres of the cloud provider.

There are several instance types that provide varying combinations of CPU, memory, storage, and networking capacity.
